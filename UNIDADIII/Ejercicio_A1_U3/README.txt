Lenguajes de Interfaz
Unidad III

Instrucciones: Realice los siguientes ejercicios en lenguaje ensamblador. 

Ejercicio 1. Escriba un programa para restar tres números enteros, usando sólo registros de 16 bits. Inserte las instrucciones necesarias para mostrar los valores de los registros. 


Ejercicio 2. Escriba un programa que contenga la definición de al menos cuatro tipos de datos. Inicialice cada variable con un valor que sea consistente con su tipo de dato. Agregar los comentarios necesarios en el código fuente para explicar cada tipo de dato.


Ejercicio 3. Escriba un programa que defina constantes simbólicas para todos los días de la semana. Cree una variable arreglo que utilice los símbolos como inicializadores. Muestre en pantalla los valores del arreglo.



Entregables:
Comprimir una carpeta que contenga los siguientes archivos. Nombrar al archivo de la forma: U3_A1_Nombre_Apellido.
Archivos de código fuente (extensión .asm o similar). Nota: Colocar a cada archivo el nombre de EjerN_U3_A1. Por ejemplo, para el ejercicio 2, Ejer2_U3_A1
Captura de pantalla de la ejecución de cada solución. Mostrar el código utilizado y el resultado en la ejecución.
